n8n 是一个公平代码（fair-code）工作流自动化平台，具备原生 AI 能力，支持可视化构建与自定义代码结合，并提供 400 多种集成方式。用户可以选择自托管或云端部署，实现灵活高效的数据流和业务流程自动化。

### 2. 核心功能
- **可视化与代码混合编辑**：结合拖拽式工作流构建器与自定义代码节点，满足从低代码到全代码的需求。
- **原生 AI 集成**：内置人工智能能力，可直接在工作流中调用 LLM 和其他 AI 服务进行智能处理。
- **广泛集成生态**：提供 400 多个预置连接器，支持与各类 API、SaaS 服务和数据库无缝对接。
- **部署灵活性**：支持自托管（Self-hosted）以保障数据隐私，也提供云端版本以便快速上手。
- **MCP 协议支持**：作为 MCP 客户端和服务器，增强与大语言模型生态系统的互操作性。

### 3. 适用场景
- **企业后端自动化**：自动同步 CRM、ERP 和数据库中的数据，减少人工重复操作。
- **AI 驱动的工作流**：利用原生 AI 功能自动生成内容、分析数据或执行智能决策流程。
- **开发者工具链整合**：通过自定义代码节点和 API 集成，连接 GitHub、GitLab 等开发工具实现 CI/CD 自动化。
- **数据管道构建**：使用可视化工具搭建 ETL 流程，轻松整合多源数据并进行清洗和转换。

### 4. 技术亮点
- **TypeScript 原生开发**：基于 TypeScript 构建，确保类型安全和良好的可扩展性。
- **公平代码许可（Fair-code）**：在保持开源可访问性的同时，限制商业用途，平衡社区贡献与商业可持续性。
- **MCP 原生支持**：率先集成 Model Context Protocol（MCP），为 AI 应用提供更标准、高效的上下文交互机制。
