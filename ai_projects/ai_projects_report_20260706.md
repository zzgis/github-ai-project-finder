### funNLP
- 1. **中文简介**
funNLP 是一个全面且丰富的中文自然语言处理（NLP）资源仓库，集成了敏感词检测、实体抽取、情感分析及各类专业领域词库。它汇集了从基础分词、词向量到基于 BERT 等前沿模型的预训练资源，涵盖了文本分类、问答系统及知识图谱构建等多种应用场景。该项目旨在为开发者提供一站式的中英文 NLP 数据处理、增强与模型训练素材。

*   **基础文本处理与清洗**：提供中英文敏感词过滤、繁简转换、停用词、反动词表及拼写检查等工具。
*   **实体识别与信息抽取**：支持手机号、身份证、邮箱等关键信息的自动抽取，以及基于 BERT 和 BiLSTM 的命名实体识别（NER）。
*   **语义分析与情感计算**：内置词汇情感值、同/反义词库、否定词库，支持文本相似度匹配、情感分析及谣言检测。
*   **领域专用知识库**：收录了医学、法律、汽车、财经、古诗词等垂直领域的专业词库及人名、地名数据库。
*   **前沿模型与数据集集成**：聚合了 CLUE、OpenCLaP、UER 等中文预训练模型资源，以及大量 NLP 竞赛数据集和基准任务。

3. **适用场景**
*   **内容安全审核系统**：利用敏感词库和暴恐词表，快速构建互联网内容的合规性检测机制。
*   **智能客服与聊天机器人开发**：借助对话语料库、意图识别模型及多轮对话数据集，训练高性能的中文对话系统。
*   **垂直行业知识图谱构建**：使用医疗、法律或金融领域的专用词库和三元组抽取工具，搭建行业专属的知识图谱。
*   **NLP 算法研究与基准测试**：作为研究人员参考，获取最新的中文预训练模型、数据增强技术及各类 NLP 任务的 SOTA 结果。

4. **技术亮点**
*   **资源极度丰富**：囊括了从传统规则方法（如正则、词典）到深度学习（BERT, GPT-2, ALBERT）的全链路 NLP 工具链。

*   **紧跟学术前沿**：及时更新并整理了清华大学 XLORE、CLUE 评测基准、各类顶会论文代码及最新预训练模型，极具时效性和参考价值。

3. **适用场景**
- 需要将模型从训练框架（如 PyTorch）迁移到部署引擎（如 ONNX Runtime）的场景。
- 在资源受限设备（如移动端或嵌入式系统）上优化和运行深度学习模型。
- 构建多框架协作的机器学习流水线，消除供应商锁定风险。

Hermes Agent 是一款智能代理工具，旨在随着用户的使用习惯不断进化与成长。它支持多种主流大语言模型（如 Claude、GPT 等），提供灵活且强大的自动化交互能力。该项目致力于通过持续学习优化用户体验，实现更自然的 AI 协作。

2. **核心功能**
*   支持集成 Anthropic Claude、OpenAI GPT 等多种主流大语言模型。
*   具备自我迭代能力，能根据用户反馈和使用数据持续优化表现。
*   提供灵活的 API 接口，便于开发者进行定制化开发和扩展。
*   支持多轮对话与上下文记忆，确保交互的连贯性与准确性。
*   兼容多种应用场景，包括代码辅助、自动化工单处理及日常问答。

3. **适用场景**
*   **个性化智能助手**：作为日常工作的个人助理，处理邮件、日程安排及信息查询。
*   **开发辅助工具**：协助程序员进行代码生成、调试及文档编写，提升开发效率。
*   **自动化工作流**：集成到企业系统中，自动执行重复性任务或数据预处理流程。
*   **多模型对比测试**：开发者可用于快速切换和对比不同 LLM 在特定任务上的表现。

4. **技术亮点**
*   采用模块化架构设计，支持热插拔不同的 LLM 后端，无需重构代码即可切换模型。
*   内置增量学习机制，能够记录用户偏好并动态调整响应策略，实现“越用越聪明”的效果。
