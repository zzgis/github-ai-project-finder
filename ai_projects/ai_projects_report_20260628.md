### Godcoder
- 1. **中文简介**
Godcoder 是一款面向桌面的本地优先开源编码智能体，强调数据隐私与安全性。用户需自备大语言模型（LLM）密钥，代码仅在本地处理，仅将必要信息发送至模型提供商，AI 甚至能构建自身的运行环境。

2. **核心功能**
*   采用本地优先架构，确保代码数据始终保留在用户设备上。
*   支持用户自行接入外部 LLM 密钥，灵活选择模型服务。
*   具备自主构建开发环境（Harness）的能力，实现自动化编码辅助。
*   基于 Rust 和 Tauri 开发，提供轻量级且高性能的桌面应用体验。
*   集成 MCP（Model Context Protocol）标准，增强与大模型的交互能力。

3. **适用场景**
*   对代码隐私和安全有极高要求的开发者或企业团队。
*   希望在不依赖特定云厂商锁定的情况下，灵活配置不同 LLM 服务的极客用户。
*   需要本地化、离线或低延迟编码辅助工具的桌面端开发者。

4. **技术亮点**
*   **技术栈优势**：结合 Rust 的高性能与 Tauri 的低资源占用，打造轻量级桌面应用。
*   **协议支持**：原生支持 MCP 协议，便于扩展上下文和数据连接能力。
*   **隐私设计**：严格的本地优先数据流设计，最小化敏感信息外泄风险。
