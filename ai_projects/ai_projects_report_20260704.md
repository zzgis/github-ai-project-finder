### learn-agent
- ### 1. 中文简介
该项目旨在从零开始构建一个能够稳定运行的 AI 编程助手，其核心机制源自真实产品 Reina，并深入剖析了 Claude Code、Codex 和 Cursor 等主流工具的底层工作原理。项目包含 15 个可运行的教学课程，且完全零依赖，非常适合希望深入理解 LLM Agent 内部逻辑的学习者。

### 2. 核心功能
*   **从零实现 Agent 循环**：通过 15 个逐步教程，演示如何构建完整的 AI Agent 运行闭环。
*   **真实机制移植**：将商业级产品 Reina 中的存活机制直接应用于代码实现中。
*   **零依赖环境**：无需安装额外库，仅使用原生 JavaScript 即可运行所有示例。
*   **主流工具原理解析**：直观展示 Claude Code、Cursor 等热门 AI 编程助手的底层交互逻辑。
*   **交互式学习体验**：提供可直接执行的代码实例，支持“边做边学”的实践模式。

### 3. 适用场景
*   **AI 开发者进阶学习**：希望深入了解 LLM Agent 架构及循环机制的工程师。
*   **编程工具原理研究**：想要探究 Cursor、Claude Code 等工具背后技术实现的爱好者。
*   **轻量级原型开发**：需要快速搭建无依赖、低开销 AI Agent 原型进行概念验证的场景。
*   **教学与培训材料**：作为讲解 AI Agent 工作机制的入门实战教程。

### 4. 技术亮点
*   **极简架构**：在零外部依赖的情况下，完整实现了复杂的 Agent 状态管理和决策逻辑。
*   **工程化实践**：不仅关注理论，更强调代码的可运行性和在实际产品中的机制落地。

### fable-soul
- 1. **中文简介**

*   **沟通优化**：规范 AI 与用户或系统之间的交互方式，使其表达更符合工程标准。
*   **层级集成**：作为独立层嵌入现有 AI 编程工作流，无需大幅修改基础模型。

3. **适用场景**
*   需要提高自动化代码生成准确性的 CI/CD 流水线。
*   构建具备自我纠错能力的智能编程助手（Copilot）。
*   对代码质量和安全性有极高要求的企业级软件开发流程。

4. **技术亮点**
*   采用“评判层”架构，将思考、验证与通信解耦，便于灵活集成到不同 AI 代理中。
