### Godcoder
- 1. **中文简介**
Godcoder 是一款面向桌面的本地优先、开源编码智能体，采用 Rust 和 Tauri 构建。用户可自备大模型 API 密钥，确保代码数据仅留在本机，仅在向模型提供商发送请求时才会短暂传输。该智能体具备自举能力，能够构建自身的运行环境。

2. **核心功能**
*   **本地优先架构**：强调数据隐私，代码文件默认保留在用户本地机器上。
*   **自带密钥（BYOK）**：支持用户接入自有 LLM API 密钥，灵活选择模型供应商。
*   **桌面应用体验**：基于 Tauri 框架开发，提供轻量级且高效的桌面端交互界面。
*   **MCP 协议支持**：集成 Model Context Protocol，增强与外部工具和数据源的连接能力。
*   **自举构建能力**：AI 智能体能够自主构建和维护其运行所需的 Harness 环境。

3. **适用场景**
*   **注重隐私的开发团队**：需要确保源代码不出域或仅受控传输的企业内部开发者。
*   **多模型灵活切换用户**：希望在不同 LLM 供应商之间自由切换以优化成本或性能的高级开发者。
*   **本地化 AI 编程助手需求者**：偏好桌面原生应用体验，而非浏览器内嵌 AI 工具的用户。
*   **Rust/Tauri 生态爱好者**：对基于 Rust 构建的高性能桌面应用及 MCP 标准感兴趣的技术人员。

4. **技术亮点**
*   **Rust + Tauri 技术栈**：利用 Rust 的性能优势与 Tauri 的低资源占用特性，打造高效安全的桌面应用。
*   **本地优先设计模式**：通过架构设计最大限度减少数据外泄风险，保障用户代码安全。
