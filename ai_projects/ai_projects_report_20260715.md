n8n 是一个具备原生 AI 能力的公平代码工作流自动化平台，支持 400 多种集成。它允许用户结合可视化构建与自定义代码进行开发，并提供自托管或云端部署选项。

### 2. 核心功能
*   **可视化与代码混合编辑**：结合拖拽式节点构建与自定义脚本，灵活满足复杂需求。
*   **丰富的集成生态**：内置超过 400 种应用和服务的连接器，实现数据无缝流转。
*   **原生 AI 能力**：深度集成 AI 功能，支持智能处理自动化流程中的关键任务。
*   **灵活部署模式**：支持自托管以保障数据隐私，也可选择便捷的云端服务。
*   **MCP 协议支持**：集成模型上下文协议（MCP），增强与大语言模型的交互能力。

### 3. 适用场景
*   **企业级业务自动化**：自动化连接 CRM、ERP 等内部系统，减少人工重复操作。
*   **AI 驱动的工作流**：利用大模型自动处理文本分类、摘要生成或智能客服响应。
*   **数据同步与 ETL**：在不同数据库或 API 服务之间建立实时数据管道和转换逻辑。
*   **开发者工具链集成**：通过 CLI 和 API 将自动化流程嵌入 CI/CD 或 DevOps 体系中。

### 4. 技术亮点
*   **公平代码许可证**：在保持开放协作的同时，允许商业免费使用个人版，平衡社区与企业利益。
*   **TypeScript 原生架构**：基于 TypeScript 开发，确保类型安全和高性能的执行效率。
*   **MCP 客户端/服务端支持**：率先支持 MCP 标准，使工作流能更高效地与外部 AI 模型交互。
