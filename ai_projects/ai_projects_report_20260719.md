### mentor
- ### 1. 中文简介
Mentor 是一款专为 AI 编码代理设计的会话洞察技能，能够读取本地 Claude Code 和 OpenAI Codex 的历史记录，并生成类似 /insights 风格的 HTML 报告。该报告详细分析用户的工作模式、时间浪费点以及具体的改进措施，旨在通过回顾性分析帮助开发者实现自我提升。它兼容 Claude Code、Codex 以及其他支持技能扩展的智能代理。

### 2. 核心功能
- **多平台历史记录读取**：自动获取本地 Claude Code 和 OpenAI Codex 的交互历史数据。
- **可视化洞察报告生成**：输出结构化的 HTML 报告，直观展示编码会话的分析结果。
- **工作流程效率分析**：精准识别用户在构建代码过程中耗时最长的环节及潜在瓶颈。
- **具体改进建议提供**：基于数据分析提供可执行的优化方案，帮助开发者提升编码效率。
- **通用代理兼容性**：作为标准化技能模块，可无缝集成到各类具备技能扩展能力的 AI 代理中。

### 3. 适用场景
- **个人开发者复盘**：Coder 在每日或每周工作结束后，通过报告回顾自己的编码习惯以优化流程。
- **团队效能提升**：技术团队利用该工具分析成员的使用模式，制定更高效的 AI 辅助编码规范。
- **新手学习引导**：初学者通过对比资深开发者的分析报告，快速掌握减少无效操作的最佳实践。
- **自动化质量监控**：集成到 CI/CD 或日常脚本中，定期自动生成开发者生产力洞察报告。

### 4. 技术亮点
- **轻量级集成架构**：基于 Python 开发，作为“技能”而非独立应用运行，对现有工作流侵入性极低。
- **跨厂商 AI 支持**：同时覆盖 Anthropic (Claude) 和 OpenAI (Codex) 两大主流生态，具有广泛的兼容性。
- **聚焦行动导向**：不仅提供数据展示，更强调“具体修复措施”，直接指向生产力提升。

*   **数据集与评测基准**：汇总各类NLP竞赛数据集、知识图谱数据、语音语料及中文语言理解测评基准。
*   **垂直领域应用方案**：包含医疗、金融、法律等特定领域的问答系统、实体识别模型及行业语料。

*   **垂直行业知识图谱构建**：借助医疗、金融等领域的专用词库和实体抽取工具，构建行业专属知识库。
