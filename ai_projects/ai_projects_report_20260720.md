### textbook-to-note
- ### 1. 中文简介
该项目能将用户的 PDF 教科书转化为支持 AI 搜索的知识库，并生成包含图表引用且带有完整出处的结构化笔记。它采用本地优先架构，旨在以极低的 Token 消耗实现高效处理。

### 2. 核心功能
*   **PDF 转结构化笔记**：自动将教科书内容转换为 Markdown 格式，并保留图表和完整引用。
*   **AI 知识库构建**：生成可供检索增强生成（RAG）使用的知识库，实现智能化的内容搜索。
*   **OCR 与解析集成**：结合 OCR 技术和 PDF-to-Markdown 工具，精准提取文本及视觉元素。
*   **低资源消耗优化**：注重 Token 效率，通过本地化处理降低 API 调用成本。
*   **生态兼容性**：完美支持 Obsidian 等笔记软件，便于知识管理与回顾。

### 3. 适用场景
*   **学术研究辅助**：研究人员或学生快速整理大量文献，建立可检索的个人学术知识库。
*   **教材数字化归档**：教师或学习者将纸质/数字教材转化为可交互、带引用的电子笔记。
*   **离线智能问答**：在没有网络或隐私敏感环境下，利用本地模型对教材内容进行精准问答。
*   **Obsidian 用户工作流**：希望将 PDF 教材无缝导入 Obsidian 并实现双向链接和 AI 搜索的用户。

### 4. 技术亮点
*   **本地优先与 Token 节约**：强调数据隐私和成本控制，避免过度依赖云端大模型的高昂费用。
*   **全要素结构化提取**：不仅处理文本，还特别注重图表（figures）和引用（citations）的保留与结构化。
*   **多工具链整合**：融合了 Claude Code、OCR、RAG 及 Obsidian 生态，形成从解析到应用的完整闭环。
- 链接: https://github.com/drpwchen/textbook-to-note
