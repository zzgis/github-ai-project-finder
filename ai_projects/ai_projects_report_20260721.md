### agents-council
- **1. 中文简介**
这是一个专为 Claude Code 设计的多智能体协作插件，旨在协调包括 Codex CLI 和 Gemini CLI 在内的多个 AI 智能体工作。它通过整合不同模型的视角，为开发者提供更全面、多元的代码分析与解决思路。

**2. 核心功能**
*   支持在 Claude Code 环境中无缝调用多种外部 AI 智能体（如 Codex、Gemini）。
*   实现多智能体并行或串行协作，以获取多样化的技术视角。
*   提供标准化的插件接口，简化多模型协同工作的配置与管理流程。
*   兼容现有的 Claude Code 技能体系，易于集成到现有开发工作流中。

**3. 适用场景**
*   需要结合不同大语言模型优势来解决复杂代码重构或架构设计问题时。
*   希望在编码过程中获得跨平台 AI 工具（如 OpenAI Codex 与 Anthropic Claude）互补建议时。
*   进行代码审查或 bug 排查，需要多角度验证解决方案正确性的场景。
*   探索多智能体协同编程模式，以提升开发效率和代码质量的实验性项目。

**4. 技术亮点**
*   实现了异构 AI 智能体的统一编排机制，打破了单一模型的能力局限。
*   基于 JavaScript 构建，轻量级且易于在 Node.js 生态中进行扩展和定制。
