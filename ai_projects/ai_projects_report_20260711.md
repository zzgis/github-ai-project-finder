### grok-build-auth
- 1. **中文简介**

*   获取 Grok Build 平台的 OAuth 访问令牌及权限。

3. **适用场景**
*   开发者需要批量或自动化工具化地接入 Grok API 服务。
*   研究人员希望简化从注册到获取 API 权限的繁琐手动步骤。
*   需要将 Grok 认证信息集成到基于 CLIProxyAPI 的内部代理或网关系统中。

4. **技术亮点**

*   提供标准化的认证数据导出接口，便于与第三方代理服务对接。
