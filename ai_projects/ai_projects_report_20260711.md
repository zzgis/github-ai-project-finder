### 2. 核心功能
*   **原生 AI 集成**：内置人工智能能力，支持在工作流中直接使用 LLM 进行智能处理。
*   **混合开发模式**：允许用户结合可视化节点搭建与自定义 TypeScript 代码，灵活扩展功能。
*   **海量连接器**：提供超过 400 种预置集成，覆盖主流 SaaS 服务和 API 接口。
*   **灵活部署架构**：支持自托管以保障数据隐私，也可选择云端服务以降低运维成本。
*   **MCP 协议支持**：原生支持 Model Context Protocol (MCP)，便于连接和管理外部 AI 模型上下文。

### 3. 适用场景
*   **企业级业务自动化**：跨系统同步数据、自动触发审批流程或生成日报周报。
*   **AI 驱动的工作流编排**：利用原生 AI 功能自动化处理文本摘要、数据分析或智能客服响应。
*   **开发者工具链集成**：通过自定义代码和 API 连接，构建复杂的 CI/CD 流水线或监控告警系统。
*   **数据管道构建**：作为轻量级 iPaaS，实时抽取、转换和加载（ETL）来自不同来源的数据。

### 4. 技术亮点
*   **公平代码（Fair-code）许可**：在保持开源精神的同时，对商业竞争场景设定合理限制，平衡社区与商业利益。
*   **TypeScript 全栈开发**：基于 TypeScript 构建，确保类型安全且易于开发者进行二次开发和插件编写。
*   **MCP 客户端与服务端支持**：率先整合 MCP 标准，使工作流能更无缝地与各类 AI 模型及外部工具交互。
