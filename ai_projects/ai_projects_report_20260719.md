### mentor
- 1. **中文简介**
Mentor 是一款专为 AI 编程代理设计的会话洞察技能，能够读取本地的 Claude Code 和 OpenAI Codex 历史记录。它通过分析开发者的工作模式，生成包含构建内容、时间损耗及具体改进建议的 HTML 风格报告。该工具旨在帮助开发者进行代码编写的回顾与自我提升。

2. **核心功能**
*   自动读取本地 Claude Code 和 OpenAI Codex 的历史会话记录。
*   生成结构化的 HTML 报告，直观展示开发者的工作流细节。
*   分析时间浪费点并提供具体的效率优化建议。
*   兼容支持技能的各类 AI 代理，如 Claude Code 和 Codex。
*   支持开发者对编程会话进行回顾性分析与自我改进。

3. **适用场景**
*   希望优化 AI 辅助编程流程、减少无效交互时间的开发者。
*   需要复盘近期编码会话以识别瓶颈和提升生产力的技术团队。
*   在使用 Claude Code 或 OpenAI Codex 时，寻求自动化性能分析报告的用户。
*   致力于通过数据驱动方式改进个人编码习惯的自我提升者。

4. **技术亮点**
*   跨平台兼容性与广泛集成能力，支持多种主流 AI 编程代理。
*   提供可视化的 HTML 输出格式，便于阅读和分析会话数据。
*   聚焦于“自我改进”与“会话分析”，填补了 AI 编程工具在事后复盘方面的空白。
